Converting a Java class to a DTO implementation

You have a class called Product in your project:

class Product {
    private long id;
    private String model;
    private int price;
    private LocalDate dateOfArrival;
    private String vendor;

    //constructors, getters, and setters
}

The DTO of the class looks like this:

class ProductDTO {
    private long id;
    private String model;
    private int price;
    
    public ProductDTO(long id, String model, int price) {
        this.id = id;
        this.model = model;
        this.price = price;
    }

    //empty constructor, getters, and setters
}

Write a function that converts Product to ProductDTO.

No input.
